# Portfólio Profissional - Atawan Cazetta

Este é o meu portfólio pessoal, desenvolvido para apresentar minha trajetória, habilidades e projetos recentes como Desenvolvedor Front-End.

O projeto destaca minha transição de carreira da área de infraestrutura de redes para o desenvolvimento de software, unindo pensamento analítico com a criação de interfaces modernas.

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estruturação semântica e acessível.
- **CSS3**: Estilização moderna utilizando variáveis, Flexbox, CSS Grid e efeitos de *Glassmorphism*.
- **JavaScript**: Lógica para interatividade e efeitos dinâmicos.
- **Font Awesome**: Biblioteca de ícones para redes sociais e contatos.
- **Google Fonts**: Tipografias *Poppins* e *Roboto Mono*.

## ✨ Funcionalidades

- **Design Responsivo**: Adaptado para uma experiência fluida em dispositivos móveis, tablets e desktops.
- **Efeito Typewriter**: Animação de texto dinâmica na seção principal (Hero).
- **Cards de Projetos Interativos**: Galeria com efeitos de hover, filtros e tags de tecnologia.
- **Seção de Contato Direto**: Links integrados para WhatsApp, E-mail e Google Maps.

## 📂 Como visualizar o projeto

1. Faça o download ou clone este repositório.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
